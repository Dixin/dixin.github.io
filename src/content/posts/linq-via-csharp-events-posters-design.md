10 [LINQ via C#](/posts/linq-via-csharp) events have been held successfully. Each event is a pure English speaking technical talk. I designed some posters for the events, with no enough time to design for each event.

## LINQ via C# part 4

![LinqViaCSharp4_Dixin_PosterDesign](https://aspblogs.z22.web.core.windows.net/dixin/Media/LinqViaCSharp4_Dixin_PosterDesign_64F76D87.jpg "LinqViaCSharp4_Dixin_PosterDesign")

In part 3, lambda expression of C# is introduced, in which audiences are very interested. So according to the feedback, in part 4 [lambda calculus](http://en.wikipedia.org/wiki/Lambda_calculus) is introduced in detail. On the poster there are a big lambda symbol and the classic lambda expression - [Y combinator](http://en.wikipedia.org/wiki/Fixed_point_combinator#Y_combinator).
